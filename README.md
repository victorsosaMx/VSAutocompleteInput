# VSAutocompleteInput

VSAutocompleteInput es un componente de entrada autocompletable simple y ligero para direcciones de correo electrónico. Proporciona una interfaz intuitiva para seleccionar múltiples destinatarios de correo electrónico de una lista predefinida de contactos.

## Características

- Sugerencias de autocompletado mientras escribes
- Añadir múltiples destinatarios
- Eliminar destinatarios con un solo clic
- Mostrar destinatarios seleccionados como etiquetas
- Diseño responsive

## Uso

1. Clona el repositorio.
2. Abre `VSAutocompleteInput.html` en tu navegador web.
3. Empieza a escribir un nombre en el campo de entrada para ver sugerencias de autocompletado.
4. Haz clic en una sugerencia para añadirla a la lista de destinatarios.
5. Para eliminar un destinatario, haz clic en la 'x' de su etiqueta.

## Personalización

Puedes personalizar fácilmente la lista de contactos modificando el array `contacts` en la sección JavaScript del archivo JS.

## Dependencias

- Bootstrap 5.0.2 (cargado vía CDN)

## Compatibilidad con navegadores

Este componente debería funcionar en todos los navegadores modernos que soporten ECMAScript 5 y superior.
